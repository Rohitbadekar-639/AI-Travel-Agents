# ✈️ AI Travel Booking System

A multi-agent AI travel planner built using LangGraph, Groq LLaMA 3.3, Streamlit, Tavily Search, and AviationStack APIs.

## 🚀 Live Demo

https://ai-travel-agents.streamlit.app

## 📂 GitHub Repository

https://github.com/Rohitbadekar-639/AI-Travel-Agents

## ✨ Features

- Multi-Agent Architecture using LangGraph
- Flight Search Agent
- Hotel Search Agent
- AI Itinerary Generation
- Final Travel Plan Generation
- Real-time Travel Recommendations
- Downloadable Travel Plans
- Modern Streamlit UI
- Cloud Deployment on Streamlit

## 🛠 Tech Stack

- Python
- LangGraph
- LangChain
- Groq (LLaMA 3.3 70B)
- Tavily Search API
- AviationStack API
- Streamlit
- GitHub

## 🤖 Agent Workflow

User Query
→ Flight Agent
→ Hotel Agent
→ Itinerary Agent
→ Final Agent
→ Travel Plan
